# Used Vehicle Market Analysis Dashboard

## Overview

This project analyzes over 550,000 used vehicle sales records using Microsoft Excel. The project demonstrates data cleaning, validation, exploratory analysis, and dashboard development techniques commonly used in data analyst roles.

## Skills Demonstrated

* Microsoft Excel
* Data Cleaning
* Data Validation
* Data Analysis
* Data Visualization
* Pivot Tables
* Pivot Charts
* Dashboard Development
* KPI Reporting

## Data Cleaning Performed

* Removed 8,539 duplicate records
* Investigated missing values and outliers
* Removed records with missing odometer values
* Removed records with missing selling prices
* Investigated and removed anomalous $1 vehicle sales
* Standardized manufacturer names for consistent reporting
* Documented all cleaning decisions in a formal cleaning log

## Analysis Conducted

* Top Vehicle Manufacturers by Sales Volume
* Average Vehicle Selling Price by State
* Average Vehicle Selling Price by Model Year
* KPI Reporting and Dashboard Development

## Dashboard Metrics

* Total Vehicles Analyzed: 540,092
* Average Selling Price: $13,690
* Highest Selling Price: $230,000
* Average Odometer: 67,492 miles
